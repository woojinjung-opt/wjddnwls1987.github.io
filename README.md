# Woojin Jung GitHub Pages Starter

This is a simple academic personal homepage built from your CV.

## Files
- `index.html` — main homepage
- `style.css` — styling
- `resume.pdf` — add your PDF CV here later (optional)

## How to publish with GitHub Pages
1. Create a repository named **YOUR_GITHUB_USERNAME.github.io**.
2. Upload `index.html` and `style.css` to the repository root.
3. (Optional) Upload your PDF CV as `resume.pdf`.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch **main** and folder **/(root)**.
7. Save.
8. Your website will be available at `https://YOUR_GITHUB_USERNAME.github.io/`.

## Easy edits
- Change text directly in `index.html`.
- Change colors and spacing in `style.css`.
- Remove phone number if you do not want to expose it publicly.
- Add LinkedIn / GitHub links in the hero button section.
